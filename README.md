# プロジェクト内の主なフォルダ構成

1. **データ（`data/`）**:
   - `raw/`: 元の生データを保存
   - `interim/`: 中間処理段階のデータを保存
   - `processed/`: 最終的な処理済みデータを保存

2. **ソースコード（`src/`）**:
   - `analyzer/`: データ分析用のモジュール
   - `data_processor/`: データ処理用のモジュール
   - `interpreter/`: モデル解釈用のモジュール
   - `trainer/`: モデル学習用のモジュール
   - `visualizer/`: データ可視化用のモジュール

3. **ノートブック（`notebook/`）**:
   - `data_analysis/`: データ分析用のJupyterノートブック
   - 個別の分析用ノートブック
